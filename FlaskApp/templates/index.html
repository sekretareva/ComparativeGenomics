Comparative Genome Analysis

Веб-интерфейс для сравнительного геномного анализа прокариот

Начать

Сравнительно-геномный анализ - трудоемкий и ресурсозатратный процесс, начинающийся с аннотации геномных данных. Популярный сервис RAST отлично подходит для этих целей, но сырые данные, полученные после аннотации не дают всех ответов. Для получения значимых результатов они подлежат дальнейшему анализу. Данное приложение поможет вам осуществить этот анализ.

Классификация

Результаты, представленные сервисом RAST после аннотирования, в виде выгрузок содержат список функций. Каждая функция имеет свое положение в системе биологической иерархии функций.

Алгоритмы приложения позволяют получить данные о соотнесении функций в рамках классификации. Часть функций, что RAST не относит к подсистемам, могут быть классифицированны по ключевым словам. В результате обработки исследователь получает обширные данные по классификации выявленных функций.

Расчеты

Количественное распределение функций в рамках биологической системы может дать полезную информацию об устройстве и функционировании микроорганизма. Данный сервис предоставит сравнительные таблицы по различным исследуемым образцам и интересующим биологическим системам.

Визуализация

Сравнительные данные - таблица с множеством измерений. Для визуализации многомерного признакого пространства используются алгоритмы, чатсо используемые в машинном обучении и биостатистике - методы снижения размерности. В результате интерактивный график позволит сравнить анализируемые объекты по их генетическим признакам.
